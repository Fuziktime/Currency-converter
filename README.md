# Currency-converter

## Автор
Кузнецов Михаил Сергеевич

## Описание
GUI-приложение для конвертации валют с использованием внешнего API.
Поддерживает сохранение истории в JSON.

## Функции
- Конвертация валют
- Использование API
- Сохранение истории
- Загрузка истории
- Валидация ввода

 ## Запуск
pip install -r requirements.txt
python main.py

## Инструкция по использованию
В верхнем окне выберите валюту, которую вы хотите конвертировать.
В окне ниже выберите валюту, в которую вы хотите конвертировать число.
Далее в поле ниже введите число.
В нижнем поле отобразится результат: ваше число - валюта, из которой вы конвертировали - валюта, в которую вы конвертировали - результат конвертации.
Также результат конвертации можно увидеть под кнопкой "Конвертировать"

## Примеры
- Ввод: 100 USD - EUR
- Результат: ~92 EUR

## Тесты
- 0 - ошибка
- -10 - ошибка
- abc - ошибка
- 100 USD - EUR - корректно
